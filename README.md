# World Happiness Report 2019 — Exploratory Data Analysis

## Overview
Exploratory analysis of the 2019 World Happiness Report dataset, examining how 
social, economic, and health indicators relate to overall life satisfaction 
across ~150 countries.

## Dataset
- **Source:** World Happiness Report 2019
- **Size:** ~150 countries, 8 variables
- **Target variable:** Happiness Score (self-reported life satisfaction)

## Analysis
- Visualized the top 10 happiest countries by score
- Compared average GDP per capita between the top 20 and bottom 20 countries
- Examined the relationship between social support and happiness score
- Identified the three indicators most correlated with happiness
- Compared median healthy life expectancy between high and low happiness countries

## Libraries
- `pandas` — data manipulation
- `numpy` — numerical computation
- `matplotlib` — data visualization

## How to Run
1. Clone the repository
2. Ensure the dataset `world_happiness.csv` is in the project directory
3. Run `main.py`
